## 项目简介

这个仓库用于整理考研复试阶段常见的机器学习基础知识，内容以：

- 核心概念理解
- 数学推导梳理
- 典型算法原理
- 选择题考点速查

为主，适合在复习阶段进行系统回顾，也适合面试前快速翻看。

## 适合怎么用

如果你正在准备复试，可以按下面的方式使用这个仓库：

1. 先看 [机器学习选择题](./机器学习选择题.md)，快速建立考点地图。
2. 再按专题阅读 [回归](./回归.md)、[决策树](./决策树.md)、[朴素贝叶斯](./朴素贝叶斯.md)、[支持向量机](./支持向量机.md)、[PCA](./PCA.md)、[HMM（隐马尔可夫模型）](./HMM（隐马尔可夫模型）.md)。
3. 面试前回到题库和重点章节，重点复习定义、公式含义、算法流程和应用场景。
